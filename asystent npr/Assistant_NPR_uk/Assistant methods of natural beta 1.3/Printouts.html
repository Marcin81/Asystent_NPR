9) Printing.
      The ability to print is available through the appropriate action from the menu “File”.
      Action “Print” - print a graph of the active tab.Graph is appropriately scaled and arranged
      on the sheet. Any actions related to printing and print settings will be performed only
      when the system is a printer (or printer drivers are installed).
9c) Printing to a file in PDF format.
        Printing a chart of the active tab to a PDF file or all of the bookmarks is possible after
        selecting the “File” menu, respectively the “Print to PDF”, “Print everything to PDF”.
        In the case of export to a file in PDF format is not necessary to have drivers
        for the printer or printer on your system.
9b) Print preview.
        Print preview is available after selecting the “File” menu action “Print preview”
        or “Print previews” for the print preview of all tabs.
 
 
9d) Print setup.
        Settings page is accessible from the “File” menu. The default is included in the auto scale
        printed or exported to PDF chart. In case of problems with printing to be off by unchecking
        the auto scale action in the “File” menu. Next we empirically choose the appropriate
        settings paper size and scale.
